# Notepad
A note taking web app, spring boot exercise. 
Live demo:  https://sheijanotepad.herokuapp.com/
Test accounts: admin/admin and user/user

List of REST api endpoints:

GET:

/api/notes, 
/api/note/{id}, 
/api/{username}/notes

POST:

/api/{username}/notes/save

PUT & DELETE:

/api/note/{id}
